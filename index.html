Analisis Security Enhancement System Terhadap Website Kampus Di Aceh Dari Serangan Cross-Site scripting (XSS)Dengan Metode Jquery From Validation

Latar Belakang

Zaman Era Revolusi Industri 4.0 ini sebagian besar aktivitas pendidikan seperti pemanfaatan informasi sudah melalui Website. Website merupakan salah satu sarana media untuk menyimpan dan menyebarkan informasi dengan adanya Website mengakses informasi mengenai perkuliahan bukanlah hal sulit.

Namun, tidak dapat di pungkiri, setiap website pasti punya ancaman keamanan, ada berbagai macam ancaman keamanan website yang dilakukan oleh peretas. Layaknya komputer, website harus diberi keamanan seperti memasang software antivirus dan beberapa keamanan lainnya. Banyaknya data dan informasi yang tersebar di website mengundang pengguna internet yang tidak memiliki hak akses untuk mendapatkan data tersebut, tentunya secara ilegal, Salah satu celah keamanan yang dilakukan adalah serangan Cross Site Scripting.

Cross-site scripting (XSS) merupakan sebuah eksploitasi keamanan di mana penyerang menyisipkan script berbahaya (biasanya Javascript) di sisi klien ke suatu halaman website. Dampak dari serangan menggunakan teknik XSS ini adalah seorang peretas dapat mengetahui isi dari database dan akan menjadi sangat berbahaya jika serangan itu terjadi. Tujuan yang paling umum dari serangan XSS adalah untuk mengumpulkan data cookie, cookie biasanya digunakan secara tidak benar untuk menyimpan informasi seperti ID sesi, preferensi pengguna atau informasi login.

Tujuan

  • Untuk melihat faktor apa saja yang mempengaruhi keamanan website kampus di Aceh

  • Untuk menganalisa Bagaimana sistem keamanan website kampus di Aceh untuk mencegah serangan XSS.

  • Untuk menguji kinerja metode jquery form validation dalam menyelesaikan masalah celah keamanan serangan XSS terhadap website kampus di Aceh.

Manfaat

  • Dapat mengetahui faktor apa saja yang mempengaruhi keamanan website kampus di Aceh

  • Memberikan informasi kepada developer website kampus di Aceh supaya lebih peka terhadap ancaman serangan XSS yang sangat merugikan pihak kampus

  • Dapat melihat bagaimana hasil pengujian kinerja metode jquery form validation dalam menyelesaikan masalah celah keamanan serangan XSS terhadap website kampus di Aceh.

Keaslian Peneletian

Pada penelitian ini penulis mengambil referensi dari jurnal Suroto S, Asman A,2021 tentang Ancaman Terhadap Keamanan Informasi Oleh Serangan Cross-Site Scripting (XSS) Dan Metode Pencegahannya dengan melakukan beberapa perombakan seperti mengubah metode jquery form validation.
